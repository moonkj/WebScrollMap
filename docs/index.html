Features

긴 페이지의 끝까지, 한눈에.

Safari 우측에 떠있는 미니맵 바로 문서 전체 구조를 파악하고 원하는 위치로 즉시 이동합니다.
🗺️

미니맵 바

페이지의 헤딩, 이미지, 링크 등 구조가 우측 바에 시각화됩니다. 분홍색 인디케이터가 현재 위치를 보여줍니다.

📌

원터치 북마크

바를 꾹 누르면 그 위치에 핀이 찍힙니다. 읽다가 잠깐 다른 곳을 확인하고 돌아올 때 편리해요.

🔍

즉시 검색

바를 더블탭하면 검색창이 열립니다. 페이지 내 모든 헤딩과 텍스트를 인덱싱해 빠르게 찾아가요.

🎨

5가지 테마

Default, Sunset, Ocean, Forest, Mono — 취향과 페이지 배경에 맞춰 미니맵 색상을 고르세요.

🔒

프라이버시 우선

수집·전송 없음. 설정과 핀은 기기 로컬에만 저장됩니다. 광고 SDK, 분석 도구 일절 없음.

⚡

경량

전체 번들 55KB 미만. 페이지 로드 영향 최소. requestAnimationFrame 기반 60fps 스크럽.

2

4

북마크
