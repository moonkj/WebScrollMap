설치 방법

  1. App Store에서 "WebScrollMap" 앱 다운로드
  2. iOS/iPadOS: 설정 → Safari → 확장 프로그램 → WebScrollMap 켜기
  3. macOS: Safari → 설정 → 확장 프로그램 → WebScrollMap 체크
  4. Safari에서 긴 웹페이지 열기 → 오른쪽에 미니맵 바 표시

사용법

  • 스크롤 — 바를 터치한 채로 위아래 드래그
  • 북마크 찍기 — 바의 원하는 위치 꾹 누르기 (약 0.5초)
  • 북마크 이동 — 바에 표시된 점 탭
  • 검색 — 바 더블탭 → 검색창 열림
  • 설정 — 주소창 옆 WebScrollMap 아이콘 클릭

자주 묻는 질문

Q. 바가 안 보여요.
짧은 페이지에서는 바가 자동으로 숨겨집니다. Safari 설정에서 확장 프로그램이 활성화되어 있는지 확인해주세요.
Q. 핀을 찍어도 사라져요.
북마크 핀은 현재 브라우저 세션 동안만 유지됩니다. Safari 탭을 닫으면 초기화됩니다(프라이버시 목적).
Q. 바 위치/두께를 바꿀 수 있나요?
Safari 주소창 옆 WebScrollMap 아이콘 클릭 → 팝업에서 위치·여백·두께·테마 조절 가능합니다.
Q. 인앱 구매가 있나요?
없습니다. App Store에서 1회 구매하면 모든 기능을 영구적으로 사용할 수 있습니다.
Q. 개인정보를 수집하나요?
어떠한 정보도 수집하지 않습니다. 개인정보 처리방침을 참고해주세요.
